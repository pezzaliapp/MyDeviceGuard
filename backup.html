📤 Esporta backup cifrato

Salva tutti i tuoi dati (eventi, alert, pairing) in un file cifrato con password. Usalo per trasferire i dati a un altro device o come copia di sicurezza.

⚠️ Se dimentichi la password il file è inutilizzabile. Non la salviamo da nessuna parte.

📥 Importa backup

Carica un file .mdg per ripristinare/unire i dati. Puoi scegliere se unire al contenuto attuale o sostituirlo.
